Proyectos de Ciberseguridad

Firewall Avanzado y Sistema de Detección de Intrusiones (IDS) para PYME

Este proyecto se centró en la implementación de una arquitectura de seguridad de red robusta para una pequeña y mediana empresa. Se configuró un firewall de última generación (UTM) con reglas de filtrado de paquetes basadas en el principio de mínimo privilegio. Además, se desplegó un Sistema de Detección de Intrusiones (IDS) basado en red (NIDS) y uno basado en host (HIDS) para monitorizar el tráfico y alertar sobre patrones de ataque conocidos y anomalías.

Esto incluyó la segmentación de la red (LAN de empleados, DMZ, y red de invitados) mediante VLANs para contener posibles brechas de seguridad y minimizar la superficie de ataque.

Auditoría de Seguridad y Análisis de Vulnerabilidades Web para Plataforma E-commerce

Se realizó un análisis de vulnerabilidades y una prueba de penetración (pentesting) de caja negra y caja gris sobre una plataforma de comercio electrónico. El objetivo fue identificar y mitigar fallos de seguridad críticos siguiendo las directrices de OWASP Top 10.

Las tareas incluyeron la explotación controlada de vulnerabilidades como Inyección SQL, Cross-Site Scripting (XSS) y configuraciones erróneas del servidor. El resultado fue un informe detallado con el nivel de riesgo de cada vulnerabilidad y recomendaciones específicas de remediación, logrando una reducción del 90% en los riesgos de exposición críticos.

Desarrollo e Implementación del Marco de Políticas de Seguridad de la Información (PSI)

Este proyecto consistió en la creación y despliegue de un conjunto integral de Políticas de Seguridad de la Información (PSI), alineadas con los estándares ISO 27001 y las regulaciones de protección de datos (ej., GDPR/LOPDGDD). Se diseñaron políticas clave, incluyendo la Política de Gestión de Contraseñas, la Política de Control de Acceso y el Plan de Respuesta a Incidentes (PRI).

La implementación incluyó sesiones de capacitación obligatorias para todo el personal y la integración de las políticas con las herramientas de gestión de identidad y acceso (IAM) de la organización.
